# pi-metaboqc 全局参数配置指南

本文档详细说明了 `pi-metaboqc` 代谢组学数据清洗与分析管线中所使用的 `pipeline_parameters.json`（或 `.yaml`）配置文件。文档列出了控制整个管线行为的所有参数的物理含义、数据类型、默认值以及允许的可选项或数值范围。

> **⚠️ 注意：** > 纯 JSON 格式不支持任何形式的注释。请勿直接在 `pipeline_parameters.json` 文件中添加 `#` 或 `//` 等注释，否则会导致解析错误。如果需要保留注释，请将配置文件保存为 YAML 格式（`.yaml`）。

---

## 1. 核心数据集构建 (`MetaboInt`)

本模块参数用于将原始丰度矩阵和元数据表映射并封装为面向对象的 `MetaboInt` 多层索引数据结构。

| 参数名              | 类型        | 默认值           | 可选值/范围                                                  | 物理含义与说明                                               |
| :------------------ | :---------- | :--------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| `mode`              | `String`    | `"POS"`          | `"POS"`, `"NEG"`                                             | 质谱扫描极性模式。仅用于报告及输出文件命名。                 |
| `sample_name`       | `String`    | `"Sample Name"`  | *任意有效列名*                                               | 元数据表中代表“样本唯一名称/编号”的列名。                    |
| `sample_type`       | `String`    | `"Sample Type"`  | *任意有效列名*                                               | 元数据表中代表“样本类型”的列名。                             |
| `bio_group`         | `String`    | `"Bio Group"`    | *任意有效列名*                                               | 元数据表中代表“生物学/临床分组”的列名。                      |
| `batch`             | `String`    | `"Batch"`        | *任意有效列名*                                               | 元数据表中代表“分析批次”的列名。                             |
| `inject_order`      | `String`    | `"Inject Order"` | *任意有效列名*                                               | 元数据表中代表“进样顺序”的列名。                             |
| `sample_dict`       | `Dict`      | *详见配置文件*   | 必须包含以Key：<br>`"Actual sample"`, `"Blank sample"`, `"QC sample"` | 字典映射，用于统一用户自定义的样本命名。需严格包含实际样本、空白样本和质控样本的固定键值。 |
| `internal_standard` | `List[str]` | `[...]`          | *有效特征 ID 列表*                                           | 内部同位素标准品的特征 ID 列表，用于绘制休哈特控制图和异常样本预警。 |
| `outlier_marker`    | `List[str]` | `[]`             | *有效特征 ID 列表*                                           | 已知异常标记物的特征 ID 列表（预留接口）。                   |

## 2. 异常特征与样本过滤 (`MetaboIntFLTR`)

本模块定义了基于缺失值 (MV)、变异度 (RSD) 和空白背景的五大特征过滤规则，以及异常样本的提示阈值。

| 参数名           | 类型    | 默认值 | 可选值/范围   | 物理含义与说明                                               |
| :--------------- | :------ | :----- | :------------ | :----------------------------------------------------------- |
| `mv_global_tol`  | `Float` | `0.7`  | `0.0` - `1.0` | **[规则 1] 全局缺失率容忍度。** 缺失比例超过此值的特征将被剔除（滤除电噪音与假阳性碎片）。 |
| `mv_group_tol`   | `Float` | `0.5`  | `0.0` - `1.0` | **[规则 2] 分组缺失率容忍度。** 在**任意单一**生物学分组中缺失率低于此值的特征即可被保留（保护高组间特异性标志物）。 |
| `mv_qc_tol`      | `Float` | `0.8`  | `0.0` - `1.0` | **[规则 3] QC 缺失率容忍度。** 在 Pooled QC 中缺失比例超过此值的特征将被剔除（低于仪器稳定检测下限）。 |
| `rsd_qc_tol`     | `Float` | `0.7`  | `0.0` - `1.0` | **[规则 4] QC 变异度容忍度。** 在 Pooled QC 中相对标准偏差 (RSD) 超过此值的特征将被剔除（定量信号极度不稳定）。 |
| `qc_blank_ratio` | `Float` | `5.0`  | `> 0.0`       | **[规则 5] QC/空白均值比界限。** 要求特征在 QC 中的均值必须达到 Blank 均值的设定倍数以上。*(注：Blank 中均值为 0 的特征绝对安全并自动保留)*。 |
| `sample_mv_tol`  | `Float` | `0.5`  | `0.0` - `1.0` | **[样本预警] 样本高缺失率阈值。** 单次进样中缺失特征比例超过此值的样本将被预警（提示可能存在堵针或降解），但不强制删除。 |

## 3. 信号校正与批次效应消除 (`MetaboIntSC`)

控制用于拟合进样随时间漂移基线的非线性回归算法及超参数。

| 参数名         | 类型           | 默认值      | 可选值/范围                                                  | 物理含义与说明                                               |
| :------------- | :------------- | :---------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| `base_est`     | `String`       | `"QC-RFSC"` | `"QC-RLSC"`, `"LOESS"`,<br>`"QC-RFSC"`, `"RF"`,<br>`"QC-SVR"`, `"SVR"` | 基线拟合估计器。可选局部加权散点平滑 (LOESS), 随机森林回归 (RF) 或支持向量回归 (SVR)。 |
| `span`         | `Float`        | `0.3`       | `0.0` - `1.0`                                                | 局部加权散点平滑的窗口跨度。仅在选用 LOESS 时有效。          |
| `n_tree`       | `Int`          | `500`       | `> 0`                                                        | 随机森林回归使用的决策树数量。仅在选用 RF 时有效。           |
| `random_state` | `Int`          | `12345`     | *任意有效整数*                                               | 随机种子，确保多线程并行和多轮测试时模型输出的绝对可重复性。 |
| `svr_kernel`   | `String`       | `"rbf"`     | `"linear"`, `"poly"`, `"rbf"`, `"sigmoid"`                   | 采用的核函数类型（推荐径向基函数 `"rbf"`）。仅在选用 SVR 时有效。 |
| `svr_c`        | `Float`        | `1.0`       | `> 0.0`                                                      | SVR 的正则化惩罚系数 (C)。仅在选用 SVR 时有效。              |
| `svr_gamma`    | `String/Float` | `"scale"`   | `"scale"`, `"auto"`, 或 `Float > 0.0`                        | SVR 的核函数系数。仅在选用 SVR 时有效。                      |

## 4. 质量评估与可视化 (`MetaboIntQA`)

控制主成分分析 (PCA)、异常值检测和统计图表的渲染策略。

| 参数名         | 类型      | 默认值       | 可选值/范围                      | 物理含义与说明                                               |
| :------------- | :-------- | :----------- | :------------------------------- | :----------------------------------------------------------- |
| `corr_method`  | `String`  | `"spearman"` | `"spearman"`, `"pearson"`        | QC 样本相关性矩阵的计算方法。                                |
| `mask`         | `Boolean` | `true`       | `true`, `false`                  | 是否对相关性热图应用上三角遮罩，使其呈现为更整洁的下三角图。 |
| `boundary`     | `String`  | `"IQR"`      | `"IQR"`, `"sigma"`, `"mean-std"` | 休哈特内标控制图的上下界计算方法（四分位距法或 3 倍标准差法）。 |
| `stat_outlier` | `String`  | `"All"`      | `"HT2"`, `"SPE-DModX"`, `"All"`  | PCA 中判定异常样本的统计算法指标（要求单一满足或同时满足）。 |

## 5. 数据标准化 (`MetaboIntNorm`)

定义在进入下游统计分析之前的最后一步标准化策略。

| 参数名          | 类型      | 默认值  | 可选值/范围                             | 物理含义与说明                                               |
| :-------------- | :-------- | :------ | :-------------------------------------- | :----------------------------------------------------------- |
| `quantile_norm` | `Boolean` | `false` | `true`, `false`                         | 是否进行独立的分位数标准化 (Quantile Normalization)。开启后，将强制忽略样本维度和特征维度设定，执行全局非线性映射。 |
| `col_norm`      | `String`  | `"PQN"` | `"PQN"`, `"TIC"`, `"Median"`, `"None"`  | **样本维度（列）标准化。** 旨在消除进样体积或细胞提取浓度的系统误差。可选PQN标准化、TIC标准化、中位数标准化等。 |
| `row_norm`      | `String`  | `"VSN"` | `"VSN"`, `"Auto"`, `"Pareto"`, `"None"` | **特征维度（行）缩放。** 旨在稳定异方差并使不同丰度量级的代谢物具可比性。可选VSN标准化、自标度化、帕累托标度化等。 |
